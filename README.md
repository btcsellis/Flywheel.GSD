# Flywheel.GSD

Personal productivity system leveraging parallel AI agents to accelerate work.

## Overview

Flywheel.GSD uses markdown files as the source of truth for:
- **To-dos** - Task tracking and prioritization
- **Instructions** - Context and guidelines for AI agents
- **Skills** - Reusable capabilities for Claude Code and Claude Cowork

A frontend application ties these components together, enabling orchestration of multiple AI agents working in parallel.

## Structure

```
/todos       - Task tracking markdown files
/instructions - Agent context and guidelines
/skills      - Reusable agent skills
/app         - Frontend application
```
